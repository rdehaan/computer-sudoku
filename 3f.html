Let's these intelligent SAT solving algorithms in action. We'll start with the first example CNF formula that you played with before.

The animation below shows how a basic version of this intelligent algorithm works. It starts by making some choices. When it runs into a conflict, it adds newly discovered clauses, and it uses these new clauses to figure out how many previous choices to undo.

(Note: There are still a lot of ways in which the algorithm can be improved. For example, it makes choices simply in order of the variable numbers. State-of-the-art SAT solving algorithms also use some intelligent strategies to make more promising choices, rather than just picking the next variable. But for the animation, we'll keep things as simple as possible.)
